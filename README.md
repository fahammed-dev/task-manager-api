# Task Manger API

### APIs Routes

- get all tasks: /api/v1/tasks
- create a new task: /api/v1/tasks
- get single task: /api/v1/tasks/:id
- update task: /api/v1/tasks/:id
- delete task: /api/v1/tasks/:id

### APIs Dependencies

- express
- mongoose
- dotenv
- morgan
- cors

### APIs devDependencies

- nodemon
